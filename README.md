## ✅ How To
> 📅 2022.06.13
1. 매주 2문제
2. 매주 월요일 `22:00` 코드 리뷰
3. **각 주차별 Directory Name** : 'Week{num}'
    - num : 2자리 숫자로
    - ex) week01
4. **Directory Name** : `{사이트명}문제번호_푼사람`
    - 백준 : BOJ2573_SB
    - 프로그래머스 : Programmers60058 (URL에 존재하는 번호 사용)
5. **Commit Message** :
   - [Solve] ✍ #BOJ 2573.미팅주선하기
   - [Solve] ✍ #Programmers 괄호변환
   - [Update] ✍️ README.md
6. 4주마다 2시간 동안 모의 코딩 테스트 진행
